simplify

  • simplify(begin: number, end: number, threshold?: undefined | number): boolean
    • Defined in script-framework/types/automation-proxy.ts:124

    Simplify the parameter curve from position begin (blicks) to position end (blicks) by removing control points that do not significantly contribute to the curve's shape. If threshold is not provided, it will be set to 0.002. Higher values of threshold will result in more simplification.
    Return true if any point has been removed.

    see

    https://dreamtonics.com/synthv/scripting/Automation.html#simplify

    Parameters

    • begin: number

      Beginning point in time to process.

    • end: number

      Ending point in time to process.

    • Optional threshold: undefined | number

      Threshold of simplification. Default to 0.002.

    Returns boolean
